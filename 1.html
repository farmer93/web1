Web browsers receive HTML documents from web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into rendered page. It procides a means to create structured documents by denoting structural semantics for text such as geadings, paragraphs, lists, links, quotes and other items.
